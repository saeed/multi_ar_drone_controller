Detailed Description

The class the represents AT commands communiction to a drone.

The class takes in the different parameters given by a CHeli instance and writes the correspoding AT command and send that command to the drone (MORE DOCUMENTATION IN ATCmd.cpp). Note that following the flow of this class needs deep knowledge of how the SDK works and the format of AT commands .
